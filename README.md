## :information_source: Application installation
- `git clone https://github.com/mathwcruz/ignite-lab-design-system.git` to clone the repository

# üé≤ Run application
To run the application on the web, follow the instructions from your terminal:
- `cd ignite-lab-design-system` and `code .`
- run `yarn` to install the dependencies of the project
- Run `yarn dev` to run the application, will open at the address `localhost:5173`
